## Installation

### Steps

1. Clone the repository:
    ```sh
    git clone https://github.com/ghtmarco/VideoTapeStore.git
    ```
2. Navigate to the project directory:
    ```sh
    cd VideoTapeStore
    ```
3. Install dependencies:
    ```sh
    flutter pub get
    ```
